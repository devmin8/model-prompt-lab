3D Football simulator

Browser-based 3D football match simulator.

The simulation should run for 3 minutes and feel like a polished football match highlights package.

It should include realistic player movement, smooth ball movement, camera transitions, scoreboard overlays, and a TV broadcast-style presentation.

01

Events

The scripted match should include the following events

  • Kickoff
  • Goal
  • Foul
  • Yellow card
  • Red card
  • Penalty
  • Full time

Reference image

03

Evaluation Plan

We will run the same prompt twice for each model.

Each output will be scored out of 10.

Evaluation criteria

  1. 1 Similarity to the reference image
  2. 2 Code organization and folder structure
  3. 3 Overall visual design and polish
    • Animation quality
    • Ball movement and match flow
    • Goal, foul, card, and penalty moments
  4. 4 Time taken to complete the task

The final score will be calculated as the average of both runs.
